##### &#x20;**E-Commerce Sales Analytics \& Forecasting**


 Project Overview

This project analyzes e-commerce sales data and predicts future sales using time-series forecasting.



 Tools Used:



\- Python

\- Pandas

\- Matplotlib

\- Prophet

\- Power BI

\- GitHub



\## Project Workflow :



1\. Data Cleaning

2\. Exploratory Data Analysis

3\. KPI Calculation

4\. Monthly Sales Trend Analysis

5\. Sales Forecasting using Prophet

6\. Power BI Dashboard Development



\## Dashboard KPIs :



\- Total Sales

\- Total Orders

\- Total Customers

\- Average Order Value



\## Dashboard Features  :



\- Sales by Region

\- Sales by Category

\- Top Customers

\- Monthly Sales Trend

\- Future Sales Forecast

\- Interactive Slicers



\## Forecasting



Future sales were predicted using the Prophet forecasting model.
